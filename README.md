# 💼 Responsive Personal Portfolio Website

A **modern, responsive personal portfolio website** built using **HTML, CSS, and JavaScript**, perfect for Full Stack Web Developers to showcase their projects, skills, and contact information.

📺 **Watch Demo**: [Responsive Portfolio Website on YouTube](https://youtu.be/cYBJsggwFig)  
🔗 **Live GitHub Repo**: [github.com/palve-2003/Responsive-Personal-Potfolio-web](https://github.com/palve-2003/Responsive-Personal-Potfolio-web)  
🔗 **LinkedIn Profile**: [Nilesh Palve on LinkedIn](https://www.linkedin.com/in/nileshpalve/)

---

## 🧠 Project Overview

This project is a **fully responsive portfolio site** designed to provide a clean and engaging way to present your:

- 👨‍💻 Skills (with animated progress bars)
- 🎓 Education
- 📁 Projects
- 📞 Contact Info

It's great for students, freelancers, or developers looking to showcase their work online.

---

## 🚀 Features

- 📱 **Responsive Design** — Built with Flexbox & media queries for full mobile responsiveness
- 🧭 **Sticky & Toggleable Navbar** — Highlights active sections on scroll
- ⌨️ **Typing Effect** — Animated developer role intro using `typed.js`
- 🎯 **Scroll Animations** — Smooth appearance of sections using custom JS
- 📊 **Skills Progress Bars** — Representing skill levels with animated percentages
- 📨 **Styled Contact Form** — Includes validation-ready form layout
- 🔗 **Social Links & Footer** — Easy navigation to your profiles
- ⏫ **Back-to-top Button** — Smooth scroll functionality

---

## 🛠️ Built With

- **HTML5**
- **CSS3**
- **JavaScript**

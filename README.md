# Responsive Personal Portfolio Website 🌐

A modern, clean, and fully responsive personal portfolio website built using **HTML**, **CSS**, and **JavaScript**. This project is ideal for **Full Stack Web Developers** to showcase their skills, education, projects, and contact details in a visually appealing and professional format.

## 🔗 Live Demo

🎥 [Watch Demo on YouTube](https://youtu.be/cYBJsggwFig)

---

## 📌 Features

- ✅ Responsive design using CSS Flexbox and media queries  
- ✅ Animated typing effect for developer role presentation  
- ✅ Scroll-based section highlighting and sticky navbar  
- ✅ Smooth scroll reveal animations  
- ✅ Skills section with animated percentage bars  
- ✅ Contact form with input validation  
- ✅ Social media links & back-to-top button  

---

## 🧱 Tech Stack

- **HTML5**  
- **CSS3**  
- **JavaScript (ES6)**  
- **Boxicons** for icons  
- **Google Fonts** (Poppins)

---

## 🚀 Usage

1. Clone the repository:

   ```bash
   git clone https://github.com/palve-2003/Responsive-Personal-Potfolio-web.git
   ```

2. Open the folder:

   ```bash
   cd Responsive-Personal-Potfolio-web
   ```

3. Open the `index.html` file in your browser or using a live server.
